# AGENTS

- Use Python 3.11+.
- Keep the agent safe by default.
- Do not add shell execution or file mutation unless explicitly requested.
- Keep repository access read-only unless explicitly asked to expand scope.
- Keep workspace files ignored by git.
- Run basic import or compile checks after modifying Python files.
- Prefer small, reviewable commits.
- Do not commit `.env` files or local workspace contents.
